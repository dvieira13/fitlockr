# FitLockr
- Virtual wardrobe app that allows users to add piece they own or want, and then create outfits from those pieces and organize their items into collections(shelves).

Full-stack TypeScript project:
- **Client**: React + Vite + TypeScript
- **Server**: Node.js + Express + TypeScript + Jest tests
- **Database**: MongoDB Atlas

**Instructions for local deployment**
- Clone the repo
- 'npm i' from root to install whole app
- 'npm run dev' from root to run whole app, and seed events in DB
- 'npm test' to run backend Jest tests
- check /server/logs/ for all log files, by day
